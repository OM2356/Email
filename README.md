# Email Detector using Java

## 📌 Description
This project is a simple Java program that detects and extracts valid email addresses from user input using Regular Expressions (Regex).

---

## 🚀 Features
- Detects email addresses from text
- Uses Java Regex
- Displays all detected emails
- Shows message if no email is found

---

## 🛠 Technologies Used
- Java
- Regex
- Scanner Class
